# Organiza-o-de-compromissos

Nome: Organização de compromissos
Objetivo: Ao agendar algum compromisso, para não se esquecer, anote ele no programa
Informações serão digitadas pelo usuário: Compromisso, data, e se preciso, o que levar
Informações que serão exibidas pelo programa: Após o usuário clicar em ver compromissos, ele mostrará todos os cadastrados
Negócios utilizados: Após o usuário registrar um novo compromisso, o programa armazena e mostra na opção 2
